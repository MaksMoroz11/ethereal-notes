# Ethereal Легковесная платформа для совместной работы: управление задачами и документация с версионированием

### Цель проекта
Создать надежную платформу для централизованного хранения, структурирования и совместной работы с текстовыми данными, исключающую потерю информации и обеспечивающую прозрачность всех изменений в документах.

__________________________________________

### Описание идеи
Ethereal — это веб-сервис, объединяющий канбан-доски для отслеживания задач и структурированную базу знаний в одном интерфейсе. Система решает проблему разрозненности данных, когда задачи ведутся в одном инструменте, а документация — в другом. Каждая запись и задача сохраняет историю изменений, что исключает потерю контекста и позволяет отслеживать прогресс без ручного контроля.

__________________________________________

### Карта сайта (Sitemap)
Структура сервиса разделена на логические блоки для удобства навигации и разработки:

**1. Информационный блок:**
* **Главная (Landing):** Презентация принципов работы системы, описание возможностей версионности и ролевой модели.
* **Тарифы:** Описание уровней доступа (Индивидуальный, Командный, Корпоративный).
* **Поддержка/FAQ:** Инструкция по использованию, правила безопасности и форма обратной связи.

**2. Авторизация и профиль:**
* **Вход / Регистрация:** Доступ к системе через подтвержденную почту.
* **Настройки профиля:** Управление личными данными, настройка уведомлений, управление активными сессиями.
* **Управление командой:** Приглашение участников, распределение ролей и прав.

**3. Функциональное ядро (Рабочая зона):**
* **Дашборд:** Общая сводка по активности, последние изменения, быстрый доступ к избранным заметкам.
* **Библиотека заметок:** Единый каталог всех доступных заметок с фильтрацией по тегам, авторам и датам.
* **Мои заметки:** Персональное пространство для создания и редактирования личных записей.
* **Совместный доступ:** Страница управления правами на конкретные заметки или папки.
* **Детальная страница заметки:** Основной редактор текста, отображение метаданных и кнопок действий.
* **История версий:** Визуальное сравнение изменений, просмотр лога действий и восстановление предыдущих состояний.

__________________________________________

### MVP (Минимально жизнеспособный продукт)
На этапе демонстрации система будет выполнять базовый цикл работы:
* **Управление доступом:** регистрация, вход, создание проекта, назначение ролей
* **Канбан-доска:** создание задач, перемещение между статусами, назначение исполнителей
* **Документация:** создание страниц, базовое форматирование, автоматическое сохранение версий
* **Журнал действий:** логирование изменений задач и документов

__________________________________________

### Направления развития (Future Vision)
* **Полнотекстовый поиск:** Быстрый поиск по содержимому всех заметок с учетом морфологии языка.
* **Экспорт и импорт:** Поддержка выгрузки данных в PDF, Markdown и DOCX, а также импорт из популярных сервисов.
* **Внешние уведомления:** Оповещения об изменениях в отслеживаемых заметках через email или мессенджеры.
* **Расширенная аналитика:** Статистика использования, активность пользователей, тепловые карты редактирования.
* **Мобильная адаптация:** Полноценное PWA-приложение для работы с заметками на смартфонах.

### Анализ конкурентной среды
Ethereal развивается в условиях насыщенного рынка, ориентируясь на устранение недостатков существующих решений:

1. **[Notion](https://www.notion.so)** - Популярная платформа для управления знаниями. 
   * *Слабая сторона:* Избыточный функционал, высокая сложность освоения, ограниченная прозрачность истории изменений в бесплатной версии.
2. **[Evernote](https://evernote.com)** - Классический сервис для заметок.
   * *Слабая сторона:* Закрытая экосистема, слабая поддержка совместного редактирования в реальном времени, отсутствие детальной версионности.
3. **[Google Docs](https://docs.google.com)** - Инструмент для совместной работы с документами.
   * *Слабая сторона:* Отсутствие структурирования в формате базы знаний, нет гибкой ролевой модели на уровне папок/проектов.

**Преимущество Ethereal:** В отличие от конкурентов, проект делает ставку на **структурированную базу знаний с встроенной версионностью**. Система предлагает прозрачный контроль изменений и точное разграничение прав доступа без перегрузки интерфейса лишними функциями, что превращает сервис из простого блокнота в инструмент для безопасной командной работы.
